# 精品出版 - 心理健康评估平台

一个采用苹果风格设计理念的专业心理健康评估平台，提供科学、准确的心理状态分析服务。

## 🎯 产品特色

- **🍎 苹果风格设计** - 采用苹果官网的极简主义设计语言
- **🧠 科学评估体系** - 基于现代心理学理论的专业评估工具
- **📊 多维度分析** - 从8个维度全面分析心理状态
- **📱 响应式设计** - 完美适配各种设备和屏幕尺寸
- **🎨 精美动画** - 流畅的交互体验和微动效
- **🔒 隐私保护** - 本地数据处理，保护用户隐私

## 🎨 设计理念

### 视觉设计
- **极简主义**：去除多余装饰，突出核心内容
- **优雅配色**：采用苹果经典配色方案
- **现代字体**：使用Inter字体，提供最佳阅读体验
- **精致动效**：细腻的过渡动画和微交互

### 用户体验
- **直观导航**：简洁明了的页面结构
- **流畅交互**：平滑的页面切换和元素动画
- **响应式布局**：自适应各种屏幕尺寸
- **无障碍设计**：支持键盘导航和屏幕阅读器

## 🧪 评估功能

### 评估维度
1. **情感反应** - 对相关情绪的处理方式
2. **态度认知** - 基本观念和认知水平
3. **沟通表达** - 相关话题的交流能力
4. **行为模式** - 实际行为表现
5. **认知水平** - 知识了解程度
6. **关系亲密** - 亲密关系表现
7. **社会适应** - 社会文化适应
8. **自我形象** - 自我认知和接纳

### 评估流程
1. **欢迎页面** - 介绍评估系统和使用说明
2. **问题回答** - 20道精心设计的专业问题
3. **进度追踪** - 实时显示评估进度
4. **结果分析** - 详细的评估报告和改进建议

### 报告内容
- **综合得分** - 0-100分的综合评估
- **等级划分** - 低/轻度/中度/重度四个等级
- **详细分析** - 基于得分的个性化分析
- **维度解读** - 各个维度的具体表现
- **改进建议** - 针对性的改善指导

## 🛠 技术架构

### 前端技术
- **HTML5** - 语义化标记，无障碍设计
- **CSS3** - 现代布局，精美动效
- **JavaScript ES6+** - 模块化开发，优雅交互

### 设计特色
- **Apple Design System** - 遵循苹果设计规范
- **Inter字体** - 苹果首选的现代无衬线字体
- **渐变色彩** - 精致的渐变和阴影效果

## 🎯 使用指南

### 开始评估
1. 访问网站首页
2. 点击"开始评估"按钮
3. 阅读评估说明
4. 开始回答问题

### 问题回答
- 选择最符合您情况的选项
- 可以前后切换题目
- 确保所有题目都已回答

### 查看报告
- 完成所有问题后自动显示报告
- 可以下载或分享报告
- 随时可以重新评估
